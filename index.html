Take the github url for your index.html file...
  • https://github.com/UserName/ProjectName/blob/master/index.html
    • ...and plug it into the link generator at rawgit.com to get a dev link like...
  • https://rawgit.com/UserName/ProjectName/master/index.html
    • To use your github as a kodi file-source, use that dev link but cut "index.html" of the end:
  • https://rawgit.com/UserName/ProjectName/master/
    • ...Note you must still use the final "/" slash!!
